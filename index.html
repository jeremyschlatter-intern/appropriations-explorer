About This Tool

This tool extracts line-item spending data from the Comparative Statement of New Budget Authority tables published in Congressional appropriations committee reports. Data is automatically extracted from committee report PDFs on congress.gov.

  • Click a subcommittee to view its line items, with tabs for each fiscal year and chamber
  • Search across all reports by program name or common abbreviations (FBI, NASA, EPA, etc.)
  • Export any table as CSV for spreadsheet analysis
  • Click the report number link to view the original source PDF

Amounts are in thousands of dollars.
